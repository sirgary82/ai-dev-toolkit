## Tips for efficient shortcut use

**Tab is your most-used key.** Most Copilot sessions involve a steady rhythm of type → review → Tab (or Esc). Get comfortable with that loop before trying anything else.

**Inline chat (`Ctrl+I`) is underused.** Select a block of code and press `Ctrl+I` to ask Copilot to modify just that selection — without leaving the editor or starting a sidebar conversation.

**`Ctrl+Enter` when you're not sure.** If the first suggestion looks off, open the completions panel (`Ctrl+Enter`) to see Copilot's other options. The best answer isn't always the first one.

**Customize if needed.** All shortcuts can be remapped in VSCode's keyboard shortcuts editor (`Ctrl+K Ctrl+S`). Search for "Copilot" to find all bindable actions.
