## What's in this toolkit?

These guides cover the full spectrum of AI-assisted development — from accepting your first inline suggestion to running parallel AI workstreams with Git worktrees.

**Guides are independent.** You don't need to read them in order unless you're following a track. Jump to whatever is most useful right now.

**No tool lock-in.** Concepts like context management and prompt engineering apply to any AI coding tool, not just the ones covered here.
