This section covers three AI coding agents that sit outside the main learning tracks but are worth knowing about. They share the same general model as Claude Code and GitHub Copilot — an AI that can read your code, run commands, and iterate in a loop — but differ in backing model, authentication, and ecosystem integration.

These are reference pages, not full guides. They cover what each tool is, how to get started, and what makes it worth reaching for.

<Callout type="note">
The tools here are complementary to what this toolkit teaches. Everything you've learned about context management, AGENTS.md, prompt discipline, and reviewing AI output applies equally when working with any of these agents.
</Callout>

## AGENTS.md works here too

The `AGENTS.md` file you set up in your repository is not a Claude Code exclusive. It is part of a shared convention:

- **Codex CLI** reads `AGENTS.md` natively — it is the primary way to give Codex project context.
- **Gemini CLI** reads `GEMINI.md` by default, but can be configured to also read `AGENTS.md` (and any other files you specify) via `settings.json`.
- **GitHub Copilot CLI** reads instruction files from `.github/instructions/` and understands your repository structure through its GitHub integration.

A well-written `AGENTS.md` in your repo gives any of these tools a useful starting point without extra setup.
